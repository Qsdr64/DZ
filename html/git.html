Git

  • git clone [url] - Клонирует удалённый репозиторий по указанному URL на локальную машину.
  • git add [файл/директория] - Добавляет файл или директорию в индекс для последующего коммита.
  • git commit -m "[сообщение]" - Создаёт новый коммит с добавленными в индекс изменениями и сообщением о коммите.
  • git status - Показывает состояние рабочего каталога и индекс, а также изменения, которые готовы к коммиту.
  • git pull - Загружает изменения из удалённого репозитория и автоматически объединяет их с текущей веткой.
  • git push - Отправляет коммиты из локальной ветки в удалённый репозиторий.
  • git checkout [ветка/коммит] - Переключается на указанную ветку или возвращает к указанному коммиту.
